    練習 - 手刻 Spring 專案 #

    • 專案目標 #

    本專案旨在幫助自己練習使用 Spring 框架。目前沒有什麼過於特別的專案目標，或者是想要透過 Spring 做出什麼東西，純最就只是想要了解 Spring 中，大部分常見的程式運作流程，包含建立控制器、服務、資料庫存取層，以及實現基本的 CRUD（建立、讀取、更新、刪除）… 等。希望能透過不使用任何輔助工具、任何外掛、任何相關的方便套件，盡可能以最原始的方式了解 Spring 的程式邏輯，進而對往後的 Spring 開發更加順利。

    • 目標 #

      • 學習如何建立一個 Spring 專案。
      • 實現基本的控制器、服務和資料庫存取層。
      • 配置和使用 Spring 的依賴注入。
      • 創建一個簡單的 Web 介面，允許用戶執行 CRUD 操作。

    • 環境設定 #

      • Java 程式開發環境：JDK 20
      • Spring 版本：spring-4.0.0.RELEASE
      • 集成開發環境（IDE）：VS Code - Visual Studio Code
      • 外掛與套件：無，什麼都沒有
