    問題：從機率工程師轉後端工程師是不是一個好決定 #

    • 使用工具：塔羅牌（線上抽牌網站）

    • 結果：

    • 解析：

    會想問這題的原因是因為公司除了機率的職缺有缺人之外，後端工程師也同樣有缺人的現象。加上我自己本身就對寫程式蠻有興趣的？比起單純的計算遊戲期望值、模擬驗證期望值，接觸一點跟後端有關的東西會再更符合我心中的工作一點，（還有一點是 …畢竟現在還是新人期，換職位對於大家來說負擔應該都不會太大？）所以就萌生了想要內部轉職的念頭了，只不過還是要問問公司裡面的老大，所以不是說想換就換，因此這題更偏向「要不要去問」這種感覺。也因為這題算是小小的是非題，所以簡單翻一張塔羅牌就可以定生死了。翻出來的牌是正位的權杖騎士，代表前進、行動、衝勁、往前走、向著目標走去、跟努力之類的意涵。算是非常明顯的「去問吧～」所以 …就等下禮拜一囉！
