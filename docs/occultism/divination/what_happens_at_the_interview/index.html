    面試會發生的情況與結果

    隨手當神棍 #

    問題：面試會發生的情況與結果？ #

    • 使用工具：紫微斗數（亂數隨機起盤）

    • 結果：

    • 解析：

    盤一跳出來，官祿丁卯的機巨自化科忌嚇了我一跳 XD。

    天機巨門都是動星，遇到化科化忌的話容易有想太多　或者是敘述太贅詞的問題。如果是後者的話還好，至於前者嘛 …畢竟是面試，想太多很容易發揮爆開，所以轉頭看向了福德宮的狀態（丁丑日月字化祿）——看來是後者，那應該就沒什麼問題了。命宮是天同獨坐亥宮，對宮天梁生科自化科 + 陀羅天馬。感覺是一個自我發揮很滿的一場面試。不太會遇到卡嘴巴、或者是大部分都回答不出來的問題。整體而言這個命財官遷我會給到 78 左右的分數，雖然還是有緊張跟結巴的成分在，但大致應該能有不錯的發揮。感覺就不用太擔心自己的問題了。

    至於主面試官的狀態：紫相坐戊辰，遷移破軍甲戌自化權，算是個蠻直接、也蠻穩重的主面試官。官祿廉貞文曲忌，顯得他很在意技術相關的問題，或者是比較要求工作上的規矩或原則之類的。四化用天機忌入我的官，所以約束或者是要求的狀況會滿重的（對我自己的體感來說）。但本身又祿入田宅照官，所以大致的要求都是對魚我們在工作上能夠更方便、更有效率的要求，也蠻合理的，看來是個好的面試官呢～疾厄跟福德也沒有往我身上沖，所以也不會很討厭我，真是太好了！

    最後簡單看了一下公司狀況：疾厄宮庚午七殺文昌祿存，對宮父母武府生年祿。光是命遷吃到武府的雙祿巡逢就明示了這是一間很有獲利能力的公司，何況命財官遷沒有煞忌做沖破。這個利潤是真的蠻好的？這就是軟體工程的實力嘛 …，果然資訊業才是毛利最多的行業啊（哈哈哈哈）。官之田本身也是祿入財福、忌入命，算是很穩紮穩打「財入我庫」的狀態，所以薪水也不用煩惱了。

    至於田宅丙寅的逆水忌嘛 …自己再注意就好。
