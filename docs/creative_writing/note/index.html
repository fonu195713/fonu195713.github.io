    「硬」的人不去做最「硬」的事

    第一話的 5 個步驟 #

    1. 介紹世界觀
    2. 告訴觀眾主角想要做什麼
    3. 交代主角的背景
    4. 讓主角經歷一些衝突
    5. 用一個巨大的變化來做收尾

    (1 + 2 + 3 + 4) = 5

    • 什麼背景的人，經歷了什麼事情，造成他怎麼樣的麻煩（或加強他哪方面的心境）
    • 可以用世界觀讓主角的背景變得既普通又特殊

    巨人：用新的坑來填補舊有的坑，讓【所.有.】的坑都不離第一個坑太遠

    獵人：開放式劇情，讓所有的故事在【當下】就全.部.講.完.

    封閉式劇情：自問自答式的劇情

    故事的開場 = 自我介紹 = 觀眾的面試

    利用特徵差異最大化來安排角色性格

    嘗試寫一個巨人的故事 #

    • 從一個最初的想法、元素、或概念來萌芽
    • 人類被圈在牆內，牆外都是充滿危險的巨人，中間有著障礙
    • 【找衝突點】
    • 全力把該「衝突點」放到最大最大
    • 把衝突點以外的「所有事情」都用來塑造那個衝突
    • 按照合理的邏輯把所有事情推理下去

    遵循故事內在（像是自由、內涵、意義、象徵、精神、「故事真正的樣子」） #

    一般情況下，故事會需要有良好節奏、循序漸進的發展，但如果角色們是怪咖時，他們的真實人生一定是相對「不工整」的，這時候，要不就像【自殺突擊隊】的作法，把它們的「怪」塞進「工整」的故事裡面，要不就【釋放角色破壞故事的權力】：比較好的做法是用角色的「特點」來破壞故事，像「瑪奇瑪把下屬當狗」，那這樣的話，即使「帕瓦」是重要角色，依然直接讓瑪奇瑪破壞掉帕瓦的重要性，讓「狗真的只是狗」。這樣就可以「讓故事遵循了瑪奇瑪的內在」，也就是「破壞了故事的平衡」和破壞觀眾對故事的看法與習慣認知。

    反邏輯正因為最懂邏輯，所以才曾將邏輯反過來運用

    1. 角色的命很重要，所以要好好珍惜
    2. 反派只會越來越強，最後的魔王在一開始就會有線索
    3. 人越害怕，惡魔就會越強

    「設定」如何使用？如何跟故事配合？ #

    1. 核心法：故事之初就把設定都說玩，後續故事都是靠最初的設定碰撞出來的（2077）

    2. 接露法：隨著故事的發展，慢慢接露所有的設定。那些設定是原本就有的，只是挑適合的時機或地方說（海賊王、巨人、鋼鍊）

    3. 增生法：隨著故事發展，越來越多新設定，但都是依循原有的設定衍生的（獵人）

    4. 憑空法 ：憑空冒出新設定，但不見得根源有的設定有關係（獵人、海賊王）

    5. 感覺法：感覺對了，就這麼做。不談來因，不談緣由（神隱少女）設定 > > > «故事»

    6. 設定要怎麼處理

    7. 故事適合哪一種設定的處理方式

    8. 想要的設定處理方式適合怎麼寫故事

    新手快速進步的方法：

    1. 快速累積創作量
    2. 如何累積
      • 有沒有很想表達的事情？試著用故事表達
      • 對覺得「明明這樣可以更好」的作品用自己的方式改寫一次
      • 想寫什麼就寫什麼，沒興趣就放掉、就斷掉
      • 找一個作品來模仿，把重點換掉，避免淪為照抄
      • 呈上點，找一個比較看得懂的作品（不要只是喜歡（ex. eve, 神隱少女））
      • 挑一個很讚的概念來模仿，而不是整個故事
    3. 不要為累積而累積，要知道累積了那些問題與疑問
    4. 模仿、模仿、和模仿
    5. 具體寫出來，不要只是腦中想

    明確主題 #

    • 要讓讀者知道你的故事想要幹嘛
    • 主題 = 一個人（或主角）想要再怎樣的環境裡面做出什麼
    • 明確的主題 =足夠理解自己想寫或想講的故事
    • 如何表達主題：
      • 在故事裡事幹嘛的？
      • 有什麼能力、習性、弱點
      • 在故事裡的意義、影響
    • 【主題】要是一個可以「討論」的狀態
    • 【主題】需要清晰呈現，才會有第一話跟公式五步驟
    • 【主題】必須因【設定】而定
    • 【設定】必須因【主題】而生

    定義情節 #

    • 情節就是『因為這樣』『所以那樣』的一個過程
    • 情節需要環環相扣，也就是需要因果關係
    • 思考情節是為了什麼而存在？
    • 要記得：情節是用來表達主題用的
    • 可以把主題和世界觀當作因，情節當作果
