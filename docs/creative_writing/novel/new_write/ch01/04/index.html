    與此同時，位於村落另一處不起眼的小屋內，一位身著深藍色連帽長袍、腰間繫著腰包、有著金色短髮、以及深藍色眼眸的精靈族少年，正坐在屋內的書桌前，並在桌上擺放著紙筆，似乎是準備要寫點甚麼東西。

    「那麼…我想要重新整理一下剛才的所有重點。」精靈少年說道。 「嗯。」回覆精靈的，是一位半躺在床上的人族男孩，看上去莫約 12-15歲，有著一頭褐色的頭髮。此外，各種纏繞於身體多處的繃帶，則說明著男孩在不久之前曾經歷過什麼重大的傷害。 「首先，我的名字叫做赫力·佴瑠斯·貝魯托爾，是公會針對這起案件所指派的調查員。然後你的名字叫做格雷瑪·伊爾，是事件的主要參與者，目前12歲，在弗洛斯村內擔任林場的伐木工人，對嗎？」說話的同時，赫力也開始動筆進行書寫，看樣子是在紀錄對話的一些內容。 「對。」格雷碼回答。 「好的。接著，你在4月15號的上午11點左右——也就是在半天之前，因為聽聞了克弗爾酒館中，一位橘髮矮人對其餘酒客的說詞，在明知規定不允許、也有可能有失去生命的情況下，仍然獨自闖入了坎澤爾森林深處的魔物巢穴，並在巢穴裡頭發現了一頭正在照顧幼鼠的雌性鑿城鼠嗎？」 「……對」 「真是個有勇無謀的舉動呢。」赫力平靜地說。 「你知道如果公會或王國的司法體系認定這是一件『可能造成效仿』的惡意舉動的話，你最多會面臨長達3年的強制兵役、或大量的罰款嗎？」沒有對說出來的話語做出任何情緒上的點綴，金髮精靈仍舊是平靜地說著。 「…不知道。」 「那你知道『不知者無罪』的這種說法是需要經過舉證，但根據目前的情況，沒有任何人可以幫你說出有力的證詞，也沒有任何物件可以幫你的行為做出的答辯，也就是說——你現在就跟一頭任人宰割的肥羊沒有兩樣嗎？」 說完話的同時，赫力暫停了手中的動作，並將視線移動到了躺在床上的格雷瑪中。雖然語調從頭到尾都很平淡，但他那銳利的眼神，與講出來的強烈措辭，還是讓格雷瑪不由得緊張了起來。 「……對不起。」 「那麼，我就當作你對剛才的描述沒有任何異議了。」將視線移回手中的文件當中，金髮精靈就像是台毫無感情的機器一般，仍然用著一貫的平順語氣說著口中的話語。 「…嗯……」 之後的一段時間，赫力沒有再對格雷碼作出任何的提問。 對於格雷瑪來說，雖然從鑿城鼠的巢穴中逃脫出來是一件再好不過的事情，不過眼下，雖然沒有生命危險，但身旁這位精靈族的金髮少年，僅是一語不發地寫下一行又一行的文字，還是讓格雷瑪不由得擔心了起來。 「你在擔心著什麼嗎？」像是看透了格雷瑪的心情一般，赫力率先打破了兩人的沉默說道。 「我之後…會怎麼樣…」 「都自己一個人踏進魔物的巢穴了，還會擔心這樣的事情嗎？」 「…嗯……」格雷瑪同樣無法說出任何的辯解。 然後又是一陣的沉默。 正如赫力所說，相對於公會可能判處的刑罰，隻身踏入魔物的巢穴對於格雷瑪可能受到的傷害是明顯更大的。畢竟公會裡的幾乎任何一個人，都不會在僅僅意識到有人靠近自己之後，就毫無理由地將一個12歲的人族男性殺死。 但對於格雷瑪來說。與其犯下「其他可能會被公會予以懲罰」的事情，「嘗試挑戰身為冒險者會做的事」才是對於他來說，更能夠獲得心目中所期望的報酬的最好方法。 但當然啦，這種想法也在今天早上被擊碎了就是。 「你是帕迦·帕托雷的孫嗣對吧？」接下來打破兩人間的沉默的，還是在此刻較為具有話語主導權的赫力。 「對，帕迦婆婆是我的奶奶。」 聽聞格雷瑪的答覆之後，赫力只是「哎…」了一聲，接著並說道：「也罷，這或許就是遺傳吧。」 「畢竟帕迦以前也做過很多類似的麻煩事就是了。」 ——誒？ 什麼意思？奶奶以前也做過很多類似的麻煩事？ 「在大約40年前吧？我們曾經以冒險者的身分組隊行動過。在某種程度上來說，這也是為什麼我現在會坐在這裡的原因。」不等格雷瑪開口問出任何問題，赫力變解釋著方才所說出來的話語。 然後，他從身上的腰包抽出了一封帶著泛黃的信件，並將之交給了身旁的格雷瑪。而後者在接下了精靈少年所給予的物件之後，便徑直地打開了信封。 只見信封裡歪歪斜斜地寫著兩行字句：可以幫我的孫子格雷瑪·伊爾成為一位合格的冒險者嗎？就當作是老朋友的最後請求。 「她不是一個會時常麻煩別人的人。」赫力說道。 「你應該是曾經和她說過說過『想成為冒險者』之類的話吧？不然我實在想不到她寫這封信的理由。」 「嗯…」赫力說得沒錯。格雷瑪在踏入鑿城鼠的巢穴之前，是一名非常非常嚮往著成為冒險者的人。 除了他因為在很早以前，就能夠透過村落的酒館接觸到其他的冒險者，並從他們的身上聽聞到關於暴險者的種種見聞外。另一個更重要的原因則是因為他的奶奶——帕迦·帕托雷在定居於弗洛斯村之前，也曾是一名登界遊方的冒險者。 作為男孩自有意識以來所接觸到的唯一親人。無論是身體還是心靈，帕迦婆婆對於格雷瑪的影響可說是幾乎無法撼動的重要。或許是出於憧憬？又或者只是盲目地追隨也說不定？總之可以確定的事情是：自從帕迦婆婆在3年前因年紀的原因去世之後，格雷瑪的內心便浮現了「我也想要當冒險者」的這種想法。 也因此，他才會在明明已經知道公會不允許未擁有冒險者執照的任何人種，踏入魔物的巢穴，仍嘗試進行這項危險的舉動——因為他想要向公會證明，自己也是有能力成為冒險者的那個人。 「明明再過4年就能夠進行冒險者的初等考核了，為甚麼這麼著急呢？」 「因為奶奶也是12歲…」 「就因為這種無聊的理由嗎？你知道扣除年資以外，冒險者的死亡率是隨著年紀成反比的嗎？」一直以來情緒皆不動於色的精靈，在此刻露出了些微可見的憤怒與嫌惡。 「扣掉40年前的環境不說，即使單論這幾年的情況，無論是任何人種，經歷2年以內的冒險者都還是有著約2-3成的死亡率，20歲以前的冒險者更是佔了這其中的6成左右。這也是為甚麼公會在之後要限制『未滿16歲的任何人種皆不得提交冒險者的入職申請』的原因。應該不難想像吧？愛惜自己的生命難道就那麼難嗎？」 「不僅奶奶，就連孫子也是善於給周遭的人帶來麻煩的性格。」 「…對不起……」面對赫力從身旁提出的一連串責罵與質問，一直以來皆處於話語下風的格雷瑪，在此刻顯露出了更加自責與受挫的表情。 「……也罷，至少就麻煩的情況來看，你做的事還算是很好善後了。」或許是意識到自己說的有點言重？也或許是出於精靈天性善良的本質？赫力在感受到了格雷瑪的道歉之後，換了一個較為和緩地語氣說道。 「…還算是很好善後？」 「在不帶有任何武器的情況下嘗試與準A級的魔物進行所謂的『和平溝通』；為了免除戰爭，在未經指揮官或其餘將領的情況下隻身闖入敵國的領地，與發起戰爭的領主進行單人對決」 「對王國的高級官員惡言相向、出言恐嚇、迫使對方簽下帶有不良利益的合約；又或者是為了讓窮苦的村民能夠食得一餐溫飽，違反多項王國、魔術師、和公會的管理條例，趁著夜色偷偷潛入富豪的家中進行大量竊盜……」 「光是回想這些…就讓我感到非常頭疼了……」看到了一直以來皆不苟言笑或板著一張臉的精靈，在此刻露出了懊惱的表情，格雷瑪那帶著慌張和緊張的思緒在此刻也稍微平緩了一些。 ——原來奶奶以前也做過類似的事情啊？ 「不要因為犯下的錯誤比較輕微，就露出那種『原來我還好』的表情好嗎？」 「好…好的！」 「閒聊結束，把話說回來。」金髮的精靈話鋒一轉，將話題又帶回到了最一開始的部分，同時說話的語氣也在此時變得較為嚴肅。 「你說你在遭遇了鑿城鼠之後，並沒有直接與之發生衝突，而是遵守著橘髮矮人的忠告，再看見了魔物之後，就緩慢而小心地離開了老鼠的巢穴，是嗎？」 「對。」格雷瑪堅定的答道。 「然後身上的傷則是在回程的路上，因為不小心腳滑，跌落到巢穴附近一個比較深的地勢而造成的嗎？」 「對。」 「那包紮的部分呢？」 「是自己在回來的時候弄的。」 「還真是熟練的技法啊…」帶著些許的懷疑，赫力看向了格雷瑪身上包裹的痕跡說道。 「畢竟從以前工作開始就小傷不斷……」 「也罷。就當作是這麼回事吧。」似乎對於格雷瑪的「解釋」沒有興趣，赫力在得到了問題的答覆之後，僅僅是說了這句話。 「那麼，就麻煩你確認一下文件上的內容，如果沒有問題的話，就在左下角的地方簽上你的全名。」說完，精靈少年便將手中一直以來在書寫的文件交給了身旁的褐髮男孩。 「好的…」 接著，因為在接過紙張之後，沒有看到紙上關於「刑罰」的描述，格雷瑪又再次小心翼翼的提問到：「那我之後…」 「通常這類的案件不會上交到公會的次級裁決機關裡面。因為像你一樣不知死活的人很多，如果這類的案件全都上交上去的話，只會徒增公會的麻煩而已。所以在通常的情況下，這類的案件都會直接以調查員——也就是我本人的調查和判定結果作為裁決的基準。」看透了格雷瑪心中的擔憂，赫力仍是用那一往平常的平淡語氣說道。 「以現在的這個情況來看，因為從偵訊的結果來說，你並沒有與魔物發生直接的衝突，巢穴內的魔物也沒有因為你的行動而變得更加暴躁或難以預測，因此在大多數的情況下，我都會以『予以口頭告誡』作為最後的裁定。」 聽完了赫力的說詞之後，像是放下了心中的大石一般，格雷瑪今天第一次在這個房間裡露出了放下心來的淺淺笑容。並在從對方接過來的文件上，簽下了自己的名字。 「只.不.過.」赫力緊接著說。 「如果再發生類似的情況，我就會以「非初犯」的情況加重處分了，這點並不會因為你是我熟人的親屬而有所寬待，還請你牢記在心，格雷瑪先生。」 「好…好的。」 「那麼，關於案件的偵訊大概就到這裡為止了。」伸出手來接過身旁褐髮少年剛簽上名的文件。金髮少年在將之收入身上的腰包之後，便站起身來說道。 「另外，關於『冒險者』的部分，除了在年紀達到16歲之後，接受公會的初等考核之外，也可以透過『經由等級4以上的冒險者予以推薦』的方式獲得資格冒險者的資格。」 「如果有你興趣的話，在明天晚上的7點25分到克弗爾酒館找我一趟。那就這樣，沒什麼事的話我就先走了。」 「好…好的！」 說完，赫力便徑直的朝向房門的位置走去。 「對了。」在右手握上門把的同時，像是突然想起什麼似的，金髮的精靈轉過頭來說道。 「以一個冒險者來說，今天早上的事，你做的很好。」 「！！！！！嗯！」
