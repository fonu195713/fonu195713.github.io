    「硬」的人不去做最「硬」的事

    第一話的 5 個步驟 #

    1. 介紹世界觀
    2. 告訴觀眾主角想要做什麼
    3. 交代主角的背景
    4. 讓主角經歷一些衝突
    5. 用一個巨大的變化來做收尾

    (1 + 2 + 3 + 4) = 5

    • 什麼背景的人，經歷了什麼事情，造成他怎麼樣的麻煩（或加強他哪方面的心境）
    • 可以用世界觀讓主角的背景變得既普通又特殊
