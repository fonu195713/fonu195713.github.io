    日記：2024/04/17 #

    不知不覺就忘記寫日記了 …

    工作之後的第 …7 天（以工作日來算的話）。這幾天同樣也發生了幾個蠻值得紀錄的事情：首先是找到新房子了～在公益路上，是一個還蠻大的社區，聽房仲跟大樓的管理員說大概總共有 800 多戶。房子前一陣子蠻搶手的，只不過因為月初發生了大地震，所以那時候震跑了一些新的租客，所以我們（我、鉦烽、和一對情侶）就補上去了。房租是以房間大小分的，因為我的房間最小，所以租金也最少（7000 only~），整體來說除了「物超所值」之外就沒有其他形容詞了。真的很感謝鉦烽找到了那麼好的地點，也很感謝她們願意用這麼便宜的價格讓我入住。

    再來是工作的進度：目前做完了 4 份老虎機遊戲的期望值試算表，開始進入到用「用程式模擬老虎機結果」的環節，但一直做得蠻不順的？可能是太久沒寫程式的關係，也可能是我還不熟 Riam 他們的程式邏輯架構，RTP 的計算一直都差了大約 3-5 % 左右，這對一個遊戲來說算是非常非常嚴重的誤差，所以一直都在嘗試修復（只是一直都沒有做得很好，所以有點小難過（哭））。想辭職的心情在經過一個禮拜的沈澱之後，大概降到了 30-25 % 左右，目前比較擔心的地方是「工作職務還沒有全部都 run 過一遍」跟「還不清楚『加班』的頻率，以及實際的工作內容」這兩項。但之後應該會慢慢適應的吧？

    再來是算命的部分：基本上算是重新把所有算命的文章跟管道都撿起來了，包含 Dcard, IG, Line 官方帳號…等，這幾天前前後後大概補了 4-5 張命盤，也賺到了一點小小的零用錢（誒嘿～），只不過因為時間比較少的關係，IG 的貼文一直都沒有辦法很準時的更新，Dcard 的貼文也是。我想要讓它們的出產率再更加固定一點，希望這點在以後搬出來住之後會有好一點的改善。（至少不用再因為通勤的關係，來回坐在機車墊上 1~1.5 個小時了）。預計六日會做大搬家，然後禮拜一就正式住進去。

    說到住進去，下一個重要的東西就是「把憨憨接回來」了。老實說我對這件事情有點小小的擔心…，一方面是不知道牠現在過得好不好？如果很好的話，那我覺得就讓牠繼續過現在的生活也不是不行（說得好像我有權利決定似的 XD）。畢竟牠都已經習慣住在那裡了，突然把牠帶過來還要重新適應，可能也會有點小小憂鬱之類的。另一方面我自己也有點害怕我是不是個不好的飼主？我希望能給牠一個很好的生活，但我不確定我能不能做到。總之目前的打算是先去問清楚憨憨的近況，如果牠過得很好、現在的飼主也很喜歡牠的話那我可能就不會接牠過來了。但如果不是這種完美情況的話，我就會從下個月開始直接升格為父親惹～

    最後寫點小小心得：有點難想像自從工作之後已經過了一個多禮拜了。對我來說，「面試」這件事情感覺就像上禮拜 3 左右才發生的事情一樣。第一天的我跟第四天的我說真的壓力是真的有接近到極限了，但好高興那時候的我有撐下來，Riam 做了 3 年才走，品諭再一個月就滿 2 年。不知道我能撐多久呢？可能就像品諭說的那樣吧？先跟自己說「撐個一年就好。」但一年對我來說好長好長 …，所以我換個方式講好了：先撐完一個禮拜就好，等一個禮拜過後再決定要不要辭職。
