    日記：2024/04/21 #

    今天是搬家的第一天。

    上次寫日記的時間是禮拜三，所以稍微記錄一下從禮拜四到今天（共計四天）的一些事情：首先是禮拜四的晚上去了一趟公司附近的超級巨星唱歌。主揪是 Peter，參加的人都是 BU01 的同事們。這次唱歌一方面是紓解壓力，另一方便也是幫 Riam 辦的小小歡送會—— Riam 只做到禮拜五，就要結束他在這裡三年的工作了。在 KTV 裡面我唱了大約 3-4 首歌，其他的時間就是吃飯、聽歌、聊天、跟坐在角落發呆。我還是有一點點小小的社交障礙，所以就不要打擾我吧（哭），讓我蹲在角落就好。

    禮拜五基本沒發生什麼事，主要的工作進度是把老虎機的模擬器寫完。大概在下午 3 點左右成功完成任務。然後就是無盡地改良、優化、跟學習 python 的新語法。下午 17:30 左右 Riam 開了一次機率部的立會，與會者有 Riam、品諭、Becky、Peter、跟我。會議的內容大致上都是在回報已經完成的東西、確認現在做的東西、還有預告即將要做的東西，比起開會，我覺得它更像是一個離開公司前的最後分享。會議結束之後，Riam 也給了我另一個老虎機的模擬練習，相較於這禮拜接觸到的，這一份新的老虎機真的是有夠複雜 …

    禮拜六同樣是還債時間，大部分醒著的時候都在做這個禮拜沒有完成的事情：像是寫 Dcard 文章、IG 發文、部落格的設計模式、密碼學、紫微斗數的內文 …等，基本上就是不停地寫寫寫、改改改。此外，禮拜六也是第一次搬家的日子，原本是打算自己把東西搬一搬的，但爸媽因為有點堅持，所以最後還是讓他們送到了公寓的門口。東西沒有很多，大部分都只是一些衣服、盥洗用具 …等生活必備品（我算是偏極簡的人）。不得不說黃埔包真的挺好用的，幾乎所有的東西都可以全部塞在那個大袋子裡面，某方面節省了我需要帶大包小包的東西的困擾。

    最後是禮拜天，早上在家裡待了一陣子之後，下午就跑來新家住了。基本上一切安好，沒有太大的問題（扣除掉我差點把衣櫃給拆掉之外 …）。鉦烽和其他兩個還不是很熟的室友人都很好，大家今天基本上都在裝飾新家，期間鉦烽還給了我一本據說是他們家祖傳的風水書籍(?)，目標是在接下來的一年之內把它看完，可以的話看懂最好。此外今天也跟 Peter 提了想要從機率改到後端的想法，過程中翻了一次塔羅，從牌上來看算蠻好的？所以就提了。結果也蠻順利的，Peter 幾乎沒說什麼就同意了，只是要經過一次後端的面試，所以能不能轉成功還是得看明後天的面試。

    再來就是比較私人的事情了，這兩天發生了一件還算蠻大的事情：憨憨死掉了。或者是說，牠可能很早就過世了，只是我這兩天才知道。搬家之前我一直都沒有確認憨憨的狀況，一方面是因為從之前表哥們的說法來看，憨憨一直都有人在照顧，所以我也沒有想太多；另一方面自己一直也有抱持著一點點的不安和不確定性？在想「還是乾脆就不要養了」所以沒有主動去問。老實說剛聽到消息的時候還蠻震驚的。隨後是難過，但很意外地，沒有到很難過。

    如果要我檢討的話，我可能一直以來都算是不合格的飼主吧？明明在大四的時候跟憨憨說「畢業之後就接他來住」，但從那之後就一直用「家裡管很嚴」當成藉口，去閃躲、逃避這件事情，前前後後拖了 2.5 年，然後就這麼過世了。我難過的點一方面是聽到這個消息，另一方面我也蠻不喜歡自己「只有難過一點點」的情緒的。總覺得我應該還要再激動一點、投入一點、沈浸在那個氛圍裡面才對。但聽到消息之後我就只是「哦 …死掉了喔 …」然後就沒有然後了。

    我覺得我是一個很失敗的飼主，在這件事情上面。這件事讓我學到「凡是如果有心，那就要趕快動起來。」就像是之前找房子一樣，擺著一臉「我沒辦法解決」的表情跟語氣問房仲姊姊可不可以幫我跟我媽說我要住這裡 …。這是我的房間ㄟ！我的人生應該要我自己決定吧？就像之前說的，我已經習慣「被寵著」的生活了，每天茶來伸手飯來張口，遇到不想做的事情就「可是我家 …可是我媽 …可是我阿公 …。」這點真的、真的、真的很不好。就像憨憨這樣，一直拖著並不能解決任何問題，反倒還會讓問題更加惡化、甚至是無法挽回。

    希望未來的我能夠引以為戒，努力變得更有責任感。
