    作品練習：主角用體內怪物開掛 #

    1. 要怎麼讓怪物進到主角體內？

      • 火影：怪物天生就在主角體內
      • 咒術：為了解救某個人，自願將怪物放入體內

    2. 什麼時機讓主角開掛會最好看？

    背景設定：

    • 主角：在競技場中獻出生命換取自由的奴隸

    • 配角：尋找強大夥伴而來到競技場的傭兵團團長

    • 配角（女）：桑提

      • 傭兵團團長，有著高超的劍術技巧
      • 善於談判、觀察、運用計謀（智力型的武將）

    • 主角一開始先不擅於 xxx

    • 讓 “壞傢伙” 一開始為了加入配角的傭兵團而展現強悍

    • 反轉：主角利用 xxx 擊敗壞傢伙，加入傭兵團，獲得自由

    • 主角追求的目標：獲得自由，離開競技場

    • 配角追求的目標：從競技場中找到有用的打手

    • 問題：如果想做序場墊高，就要讓某個強而有力的人看出主角的價值

    • 那個人會是誰？價值是什麼（戰鬥能力 …之類的）

    • 情節 1：配角和主角偶然路過，配角發現到主角的 “價值”

    • 情節 2：配角因為 “情節 1” 所以打算前往競技場一探究竟

    • 情節 3：因為某個原因，壞傢伙必須跟主角在競技場中打一架，然後主角打贏壞傢伙

    • 善用 “任務角色”（壞傢伙），把所有的事情都丟給他

    • 五步驟：世界觀、主角要幹嘛、主角背景、衝突、反差做結尾

    • 關於主角背景：讓壞傢伙作為 “主角會變成奴隸的原因（或原因之一）”

    • 衝突：壞傢伙跟主角打架

    • 反差：讓主角打贏壞傢伙

    • 介紹世界觀：（還沒想好，但這邊應該不會給壞傢伙）

    • 主角要幹嘛：剛剛說過了，主角想自由

    • 情節 4：配角告知主角 “能打贏壞傢伙的方法，就是運用體內的力量”
