    第一話的幾個重點 #

    1. 世界觀描述

      • 在這個充滿魔物的世界
      • 上古的人類為了生存，必須不斷地逃離、抵抗這些魔物
      • 直到有一天，人們發現了「火」的存在
      • 人們可以用它來驅趕魔物、擊退魔物
      • 因此，人們圍繞著「火」建立起了自己的家園
      • 隨著時間的過去
      • 人們的家園愈發茁壯、龐大
      • 「火」開始無法完全覆蓋、保護他們的家園
      • 因此，為了尋找其他的「火」
      • 人們組建了一支又一支的冒險隊伍
      • 期待可以找到新的「火」，以供人類使用、居住
      • 並稱呼這些隊伍為點燃「火」的「火種」

    2. 主角想幹嘛

    3. 主角的背景

      • 作為階級底層的主角，生活在「火」的最外圍
      • 也因為最外圍的「火」最為微弱，因此容易遭遇魔物侵擾
      • 對主角來說，活下去的最好方法便是成為「火種」

    4. 過程的衝突

      • 最外圍的生存地位最為低落、存在無理和歧視
      • 容易遇到魔物的侵擾，資源非常有限

    5. 變化做結尾

      • 主角在最後成為了最為弱小的「火種」

    • 在疲憊的生活裡，「信仰」的存在價值是什麼？
