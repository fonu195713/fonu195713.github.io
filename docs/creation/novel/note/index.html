    第一步：序場（開頭定調） #

    • 由一群地位更高的人、或更強的人，來「鋪墊」主角群
      • 有一位專門為政府解決麻煩的「清道夫」
      • 他接下來想要解決的並不是某個反抗組織的重要成員
      • 而是在某個窮鄉僻野的主角
    1. 序場是故事的定調
    2. 幫角色添加反差

    • (接續上面的內容)

      • 有一位專門為政府解決麻煩的「清道夫」
      • 他接下來想要解決的並不是某個反抗組織的重要成員
      • 而是在某個窮鄉僻野的主角
      • 因為比起反抗組織的成員
      • 主角的「天賦」在未來可能更加棘手

    • 往下拉，展現主角的弱

    • 「強的人」看重「弱的人」的哪種能力？

    • 從那個能力相互呼應

      • 清道夫看上的天賦是主角「過人的魔力性質」
      • 讓主角在登場時「魔力很差」
      • （原因為何？後面再來思考。要如何表現？也先放著）

    • 讓配角來呼應主角

      • 主、配角因為某件事相遇
      • 在事件的其中一方受到對方幫助
      • 因此有了交集。
      • 配角登場方式，要讓主角又弱又有很爛的魔力

    • 思考各種可能性

      • 主角為了想當「冒險者」在「冒險者工會」裡測試實力
      • 但測試的結果並不能被公會的鑑識人員獲得認可
      • 尤其主角的「魔力」在測試上遠低於常人的平均值
      • (但鑑識結果因為 …(還沒想到) 的原因，被清道夫給拿到了)
      • (所以才有了最一開頭「清道夫覺得主角能力麻煩」)
      • 在那之後，主角為了證明自己「並不弱」，在公會的大廳嘗試與人較量

    • 完整化

      • 配角是一個「善於使用『天屬性』魔法」的魔法使
      • 來到了主角所在的偏遠城鎮，想尋找能傳承自己魔法技術的「弟子」
      • 結果一踏進冒險者公會的大廳
      • 就看到被其他人用魔術痛扁倒在地上的主角
      • （把配角設定的特別弱或特別『強』）

    • 更加完整

      • 配角除了是一個善於使用『天屬性魔法』的魔法師
      • 也是有著通過冒險者公會「等級 5」能力鑑定的佼佼者
      • 但也因為能力過強，加上一些之前的行徑，因此也被清道夫給盯上
      • 為了躲避清道夫，也為了傳承自己的技術
      • 他在一次使用了魔術『預知』之後
      • 得到了「某東方的城鎮，可能有著之後會比自己還要厲害的人物」的情報
      • 因此開始往東方尋找，但結果不盡人意
      • 一路經過的城鎮從繁華到窮困、從熱鬧到冷清，都沒找到適合的人物
      • 然後，到了他打算放棄之前的最後一個城鎮時
      • 配角一進到城鎮的冒險者工會，就看到被其他人用魔術打趴的、倒在地上的主角
      • （… 中間故事跳過）
      • 主角的目標：想要成為「冒險者」，劍士、弓箭手、法師都行
      • 配角的目標：想要躲避「清道夫」，隱姓埋名，並同時找尋一個可以傳授的「弟子」
      • 共同的目標：學習魔法，成為一個合格的冒險者（合格的弟子）

    • 磨合的橋段

      • (這邊可以先略過，最彈性)

    • 被騙的事情，要能幫助主角做反轉

    • 妖尾的最後面

      • 配角追求的東西跟主角有關
      • 但卻被誤導，陷入困境
      • 再由真正有關的主角出面
      • 幫配角解決困境
      • 之後在收尾接回開頭
    1. 配角遭遇的困境，要有助於主角發揮
    2. 一開始為何弱？要解釋清楚，邏輯通順
    3. 收尾接回開頭，可以再持續墊高，或針對開頭丟出的東西，多丟一點東西
